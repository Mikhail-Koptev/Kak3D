4. Цветная модель

Пришло время раскрасить модель. Для этого можно

использовать материалы или использовать текстуру.

Оба способа довольно простые. Только текстуру надо

в прямом смысле рисовать, поэтому способ с текстурой

занимает больше времени. При UV-развёртывании, т.е. после

создания изображения текстуры модель становится чёрной.


Примечание: для лучшего результата от текстуры, можно при

UV-развёртывании указать расстояние между частями текс-

туры 0.03.
